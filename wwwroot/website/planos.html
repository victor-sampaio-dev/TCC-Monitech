Perguntas frequentes

Preciso comprar o Dispositivo Monitech para usar?
Sim. O plano controla o acesso ao software, mas você precisa do hardware (Dispositivo Monitech) para fazer as leituras. Compre o dispositivo aqui.
Posso cancelar quando quiser?
Sim. Sem fidelidade, sem multa. Cancele a qualquer momento e seu plano continua até o fim do período pago.
O que acontece com meus dados se eu cancelar?
Sua conta volta para o plano gratuito automaticamente. Você continua com acesso ao último dispositivo cadastrado e aos últimos 7 dias de histórico.
Posso usar um único dispositivo em mais de uma residência?
Não. Cada Dispositivo Monitech é vinculado a uma residência específica. Para monitorar duas casas, você precisa de dois dispositivos.
