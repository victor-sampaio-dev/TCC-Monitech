Perguntas Frequentes

O MONITECH funciona sem um ESP32 conectado? ▼
Sim. Você pode navegar pelo painel, cadastrar cômodos, dispositivos e configurar alertas sem nenhum hardware conectado. Os medidores e gráficos ficarão em modo de espera até que um dispositivo transmita dados.
Meus dados ficam salvos na nuvem? ▼
Não. O MONITECH é uma aplicação local — todos os dados ficam no seu próprio dispositivo (localStorage do navegador) e na sua rede local. Nenhuma informação é enviada a servidores externos.
Qual a diferença entre WebSocket e HTTP Polling? ▼
WebSocket mantém uma conexão persistente e bidirecional — o ESP32 envia dados automaticamente a cada 2 segundos, com latência mínima. HTTP Polling faz requisições GET periódicas; é mais simples de configurar e funciona em redes com restrições de portas, mas tem latência levemente maior.
Posso usar outro sensor além do PZEM-004T? ▼
Sim. O painel aceita qualquer fonte de dados que envie o JSON no formato esperado (volts, amps, watts são os campos obrigatórios). O sensor ACS712 é uma alternativa comum, assim como sensores DIY com qualquer microcontrolador que suporte Wi-Fi.
O Serial/USB funciona em qualquer navegador? ▼
Não. A Web Serial API é suportada apenas no Google Chrome 89+ e Microsoft Edge 89+. Firefox e Safari não implementam esta API. Para os outros protocolos (WebSocket, HTTP, MQTT), qualquer navegador moderno funciona.
Como descobrir o IP do ESP32 na minha rede? ▼
Após gravar o firmware e conectar o ESP32 à rede Wi-Fi, abra o Serial Monitor do Arduino IDE (115200 baud) e aguarde o boot. O IP local será impresso na saída serial assim que a conexão for estabelecida.
